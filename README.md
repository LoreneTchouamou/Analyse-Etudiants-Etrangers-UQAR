# Analyse-Etudiants-Etrangers-UQAR
Analyse exploratoire des parcours académique des étudiants étrangers a l'UQAR. Projet réalisé avec Python(Pandas) et Power BI, basé sur la méthodologie Google Data Analytics.
## Présentation du projet
Ce projet d'intégration (Session Été 2025) analyse les facteurs influençant la réussite et l'intégration des étudiants internationaux à l'UQAR. À partir de données brutes collectées, j'ai transformé l'information en insights exploitables pour proposer des pistes d'amélioration à l'université.

## Stack Technique & Compétences
- **Langages :** Python (Pandas, Matplotlib, Seaborn)
- **Outils BI :** Power BI (Tableaux de bord interactifs)
- **Environnement :** Jupyter Notebook
- **Méthodologie :** Google Data Analytics (Nettoyage rigoureux, analyse de corrélation, Data Storytelling)

##  Contenu du dépôt
- `Nettoyage_Donnees.ipynb` : Code Python utilisé pour transformer les données brutes (Excel sale) en données prêtes à l'analyse (Excel propre).
- `Rapport_Final.pdf` : Analyse complète de 15-20 pages incluant les recommandations.
- `Dashboard_Etudiants.pbix` : Le fichier source Power BI.
- `Donnees/` : Les fichiers Excel avant et après traitement.

## Résultats clés
- Traitement d'un échantillon de **55 répondants** et **23 variables**.
- Identification de liens entre la session d'entrée (Hiver/Automne) et les défis académiques.
- Création d'indicateurs clés (KPI) sur les taux d'échec et d'abandon selon le pays d'origine et le cycle d'étude.

Ce projet démontre ma capacité à gérer un cycle de données complet, de la collecte à la recommandation stratégique.
